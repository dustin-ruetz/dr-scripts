# dr-scripts

Adapted from Kent C. Dodds' [kcd-scripts](https://github.com/kentcdodds/kcd-scripts/) package and ["How toolkits (like react-scripts) work"](https://www.youtube.com/watch/?v=ZWfY-v1DCTE) video.
